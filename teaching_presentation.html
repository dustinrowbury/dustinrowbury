CSS Selectors: attributes, children, descendants

Selector

There are 51 different selectors in CSS

Type Selectors

  • Type selectors select the tag type
  • Example: p{color: blue;


Class type

  • You can give things a class by saying. Classes are case sensetive. class="class1"
  • Example: .class1{ color: red;}


ID Selector

  • You can give things an ID by by typing id and assigning them. Unlike classes you can only have one id with the same name. id="rocket"
  • Example: #rocket{ color: green;}


Universal Selector

  • Plain and simple it selects everything.
  • Example: *{ color: green;}


Decendant Selector

  • You can make a selecton more specific by naming decendents by doing this you are using decendant selectors. The ul in the example below means that this will only select a p tag if it is a decendant of a ul tag
  • Example: ul p{ color: green;}


Child Selector

  • The same as the descendant selector except you can only go back one geration id="rocket"
  • Example: div > p{ color: orange;}


Attributes Selector

  • This one looks for attributes
  • Example: a[target]{ color: purple;}
